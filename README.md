A curated list of awesome publicly available datasets published by or relevant to the UK government with the goal of making these resources more accessible to the broader data community. This repo is inspired by [awesome-public-datasets](https://github.com/awesomedata/awesome-public-datasets) and maintained by [i.AI](https://github.com/i-dot-ai).

## Contributing

To add a new dataset:

1. Create a new YAML file in the appropriate `datasets/` topic folder (or create a new folder if needed).
2. Follow the YAML specification in `dataset-spec.yaml`.
3. Commit and push your changes.
4. When merged to main, the GitHub Action will automatically update the README with your new dataset.

More details on contributing can be found in the [CONTRIBUTING.md](CONTRIBUTING.md) file.
